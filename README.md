# hackpad
this is my hackpad that i created with help from the hackclub with the blueprint program
i made hackpad for hardware monitoring and to use as a soundboard.
it has a display for monitoring a roteryencoder for volume and 4 butons for macro's like passwords en shortcuts like ctrl+c 
it does at this moment only show a foto on the screen the software for hardware monitoring the volume works

how to use
you can just plug it in after making the pcb and flash the firmware and start using it for things like change volume 

why dit i make this macropad 
i made it because i wanted to know how to make my own pcb's and make firmare, this was a nice introduction into making my first pcb 
and i also dit it of course for making my workflow better so can change and see things easyer when i am doing homework or modeling
